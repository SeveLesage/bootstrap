Qu'est ce que Bootstrap

Le Framework Bootstrap

Bootstrap est un framework (cadriciel) pour les language CSS et JS. Il va faciliter l'utilisation de ces langages en rendant notre site agréable visuellement et graphiquement De plus, ce cadriciel va faciliter le développement d'un site responsive

Installation

  • L'installation en CDN (ligne de commande) => c'est la méthode que l'on utilise sur cette page. On apppelle les librairies hébergees en ligne grâce aux liens de Bootstrap (CSS dans l'entete entre les balises meta et le title et JS dans body avant la fermeture de body et HTML)
  • L'installation sur l'ordinateur en local, voir la page avec Bootstrap en local
L'utilisation de Bootstrap en local se fait lorsque l'o développe un site ou que notre connexion internet peut se révéler faible.

Utilisation

Ici Bootstrap fonctionne grâce à ses bibliotheque remplies de CSS. On va récupérer dans les bibliotheques existantes les classes.

On déclare les classe dans le HTML, attention, si vous déclarez les classes en id, elles ne fonctionnement pas

Mettez dés maintenant le site Bootstrap en favori, ça doit devenir votre bible. Si votre classe ne fonctionne pas, il faut retourner dans la doc et vérifier l'orthographe

Pour les îcones de Bootstrap, le fonctionnement sera le même et la documentation de BS sur les îcones est compléte : n'hésitez pas à la lire

Attention !

Avant de vouloir modifier un exemple, il faut bien que vous compreniez à quoi correspondent chacunes des classes utilisées, pour être sûr de ne pas modifier et casser le comportement de l'élément.

Lexique

Cadriciel
Cadriciel est la traduction directe de framework : un logiciel qui cadre l'utilisation d'un langage (i.e. Bootstrap pour le CSS // Symphony pour le PHP)
CDN
CDN signifie Content Delivery (ou dans certains cas Distribution) Network : réseau de distribution de contenu. On appelle aussi cette façon de faire la ligne de commande. On va en fait récupérer du contenu hébergé en ligne grâce à des liens
